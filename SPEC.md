# MiniMax 账单可视化 Web 应用规格文档

## 1. 项目概述

- **项目名称**: MiniMax Bill Visualizer
- **项目类型**: 单页 Web 应用 (SPA)
- **核心功能**: 拖入 CSV 文件自动解析并可视化 MiniMax API 消费账单
- **目标用户**: MiniMax API 开发者、运维人员

---

## 3. 功能规格

### 3.1 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| CSV 拖拽解析 | 拖入文件自动解析，支持 .csv 格式 | P0 |
| 数据验证 | 校验文件格式是否正确 | P0 |
| 概览统计 | 显示总金额、Token、调用次数等 | P0 |
| 消费趋势图 | 按小时/天/周展示消费趋势 | P0 |
| 接口分布 | 饼图展示各接口占比 | P0 |
| 模型对比 | 柱状图对比各模型用量 | P1 |
| 数据表格 | 分页展示消费明细 | P1 |
| 筛选过滤 | 按日期、接口、模型筛选 | P1 |
| 导出功能 | 导出筛选后的数据为 CSV | P2 |

### 3.2 数据处理

```typescript
interface BillRecord {
  account: string;           // 消费账号
  apiKeyName: string;        // 接口密钥名称
  apiName: string;           // 消费接口
  model: string;             // 消费模型
  amount: number;            // 消费金额
  amountAfterCoupon: number; // 代金券后消费金额
  inputTokens: number;       // 输入token数
  outputTokens: number;      // 输出token数
  totalTokens: number;       // 总token数
  timestamp: string;         // 消费时间 (小时)
  result: 'SUCCESS' | 'FAIL'; // 消费结果
}
```

### 3.3 页面状态

1. **初始状态**: 拖拽上传区域
2. **解析中**: 加载动画
3. **数据展示**: 仪表盘视图
4. **错误状态**: 格式错误提示
